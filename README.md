# 🏠 Visualizing Housing Market Trends

## 📌 Project Overview

This project focuses on analyzing **housing market trends and factors affecting house prices** for **ABC Company** using data visualization techniques.

The system is built as a **Flask-based web application** that integrates **interactive Tableau dashboards and stories**. Users can explore housing data insights through a **custom web interface**.

📊 The main goal is to identify **patterns, trends, and relationships in housing data** to support **better pricing strategies and business decision-making**.

---

## 🎯 Objectives

- 📈 Analyze housing **price trends**
- 🏡 Understand the **impact of renovations on property value**
- 🔍 Study the relationship between **house features and pricing**
- 💡 Provide **data-driven insights** for strategic decisions

---

## 🛠 Technologies Used

| Technology | Purpose |
|------------|---------|
| 🐍 Flask | Web application framework |
| 📊 Tableau | Interactive dashboards & data visualization |
| 📂 Housing Dataset | Data analysis |
| 🌐 HTML / Web Interface | Display dashboards |

---

## 📂 Key Dataset Features

The dataset contains important housing attributes such as:

- 💰 **House Price**
- 🛏 **Bedrooms**
- 🚿 **Bathrooms**
- 🏢 **Number of Floors**
- ⏳ **House Age**
- 🔧 **Renovation Details**
- 🏗 **Basement Area**

---

## 📊 Dashboard Insights

The Tableau dashboard provides several interactive visualizations:

### 1️⃣ Overall Data Overview
Displays:
- Total housing records
- Average house price
- Total basement area

### 2️⃣ Sales by Years Since Renovation
📉 A **Histogram** showing how renovation timing influences house sales prices.

### 3️⃣ House Age by Renovation Status
🥧 A **Pie Chart** comparing **renovated vs non-renovated houses** based on age groups.

### 4️⃣ House Age by House Features
📊 A **Bar Chart** analyzing house age based on:
- Bathrooms
- Bedrooms
- Floors
