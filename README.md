# Visualizing-Housing-Market-Trends

## Project Overview

This project analyzes factors affecting house prices and sales trends for **ABC Company** using housing data visualization.
It is a **Flask-based web application that embeds and displays interactive Tableau dashboards and stories**, allowing users to explore data visualizations through a custom web interface.

The goal is to identify patterns in the housing market and support better pricing and business decisions using **Tableau dashboards**.

## Objectives

* Analyze housing price trends
* Understand the impact of renovations on prices
* Study relationships between house features and price
* Provide insights for strategic decision-making

## Technologies Used

* Flask
* Tableau
* Data Visualization
* Housing Dataset

## Key Dataset Features

* House price
* Bedrooms and bathrooms
* Number of floors
* House age
* Renovation details
* Basement area

## Dashboard Insights

The Tableau dashboard includes the following visualizations:

1. **Overall Data Overview** – Shows total records, average house price, and total basement area.
2. **Sales by Years Since Renovation** – Histogram showing how renovation timing relates to sales price.
3. **House Age by Renovation Status** – Pie chart showing renovated vs non-renovated houses by age group.
4. **House Age by House Features** – Bar chart analyzing house age based on bathrooms, bedrooms, and floors.

## Conclusion

This project helps identify housing market patterns and provides insights that support better real estate pricing and decision-making.
